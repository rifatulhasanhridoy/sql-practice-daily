# SQL Daily Practice

Practicing SQL daily using a retail e-commerce schema (SQL Server).

## Topics Covered
- Basics: SELECT, WHERE, ORDER BY, GROUP BY
- Joins: INNER, LEFT, SELF JOIN
- Subqueries & EXISTS
- CTEs & Window Functions (RANK, ROW_NUMBER, LAG)

## Schema
Based on a retail business: customers, orders, products, payments, reviews.
